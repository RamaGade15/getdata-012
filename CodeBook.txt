This code book is for use with -

1) the original instructions available with the dataset and 
2) full description available at  http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones 

R scipt runanalysis.R is created to process the original datasets and create a tidy dataset with file name x_tidy_summary.

This dataset has 180 rows (6 activities x 30 Subject). Following are the colum names is this data set. 

Each of the colums represent mean values of the corresponding measurement sumarized as one observation for each activity, subject combination.

Note that only mean and standard deviation are extracted and summarized.
